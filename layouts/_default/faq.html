📦 安装问题

Windows Defender 一直报毒怎么办？

+

这是常见的误报问题，您可以通过以下方式解决：

  1. 添加到白名单：打开 Windows Defender，进入"病毒和威胁防护" > "管理设置" > "添加或删除排除项"，将 {{ .Site.Params.app.name }} 的安装目录添加到排除列表。
  2. 重新下载：确保从官方 GitHub 页面下载最新版本。
  3. 验证文件完整性：使用 SHA256 校验和验证下载的文件是否完整。
  4. 临时关闭实时保护：在安装过程中临时关闭 Windows Defender 的实时保护功能。

提示：{{ .Site.Params.app.name }} 是完全开源的软件，您可以在 GitHub 上查看源代码以确认其安全性。

macOS 提示"无法打开，因为无法验证开发者"怎么办？

+

这是 macOS 的安全机制，解决方法如下：

  1. 右键点击 {{ .Site.Params.app.name }} 应用，选择"打开"
  2. 在弹出的对话框中点击"打开"
  3. 或者进入"系统偏好设置" > "安全性与隐私" > "通用"，点击"仍要打开"

对于 Apple Silicon Mac：

  • 确保下载的是 Apple Silicon 版本
  • 如果仍有问题，可以在终端中运行：sudo xattr -rd com.apple.quarantine /Applications/HiddifyNext.app

Android 安装时提示"未知来源"怎么办？

+

需要允许从未知来源安装应用：

  1. 进入"设置" > "安全" > "未知来源"，开启该选项
  2. 或者在安装时，系统会提示允许该来源，点击"允许"
  3. 推荐：直接从 Google Play 商店下载，避免此问题

🌐 连接问题

连接成功但无法上网怎么办？

+

请按以下步骤排查：

  1. 检查订阅状态：确认您的订阅是否过期或流量是否用完
  2. 切换节点：尝试连接其他服务器节点
  3. 测试延迟：进行延迟测试，选择延迟较低的节点
  4. 检查系统代理：确认系统代理设置是否正确
  5. 重启应用：完全关闭并重新启动 {{ .Site.Params.app.name }}
  6. 检查防火墙：确保防火墙没有阻止应用的网络访问

注意：如果所有节点都无法使用，请联系您的服务提供商确认服务状态。

连接速度很慢怎么办？

+

优化连接速度的方法：

  1. 选择最佳节点：使用延迟测试功能，选择延迟最低的节点
  2. 更换协议：尝试不同的协议类型（如 Reality、VLESS、Hysteria）
  3. 调整 DNS：在设置中更换 DNS 服务器（推荐 1.1.1.1 或 8.8.8.8）
  4. 关闭其他应用：关闭占用网络带宽的其他应用程序
  5. 检查本地网络：确认本地网络连接稳定

频繁断线重连怎么办？

+

解决断线问题的方法：

  • 更换节点：当前节点可能不稳定，尝试其他节点
  • 检查网络环境：确认本地网络连接稳定
  • 调整重连设置：在应用设置中启用自动重连功能
  • 更新应用：确保使用最新版本的 {{ .Site.Params.app.name }}
  • 联系服务商：如果问题持续，联系您的服务提供商

⚙️ 配置问题

如何导入订阅链接？

+

导入订阅链接的详细步骤：

  1. 复制服务提供商给您的订阅链接
  2. 打开 {{ .Site.Params.app.name }}，点击主界面的"+"按钮
  3. 选择"从剪贴板导入"或"从 URL 导入"
  4. 粘贴订阅链接，点击"添加"
  5. 等待配置文件下载完成

支持的订阅格式：

  • 标准 V2Ray 订阅
  • Clash 订阅
  • 单节点链接（vmess、vless、trojan 等）

如何设置分流规则？

+

设置分流规则可以让不同的应用使用不同的代理策略：

  1. 进入 {{ .Site.Params.app.name }} 的设置页面
  2. 找到"路由规则"或"分流设置"
  3. 选择预设规则或自定义规则
  4. 常用规则包括：
    • 国内直连：中国大陆网站直接访问
    • 国外代理：海外网站通过代理访问
    • 广告拦截：屏蔽广告域名

详细教程请参考：应用分流规则设置指南

如何设置开机自启动？

+

设置开机自启动的方法：

Windows：

  1. 打开 {{ .Site.Params.app.name }} 设置
  2. 找到"常规设置"或"系统设置"
  3. 启用"开机自启动"选项

macOS：

  1. 进入"系统偏好设置" > "用户与群组" > "登录项"
  2. 点击"+"添加 {{ .Site.Params.app.name }}
  3. 或在应用设置中启用自启动选项

⚡ 性能问题

应用占用内存过高怎么办？

+

优化内存使用的方法：

  • 重启应用：定期重启 {{ .Site.Params.app.name }} 释放内存
  • 减少订阅数量：删除不必要的订阅配置
  • 关闭日志：在设置中关闭详细日志记录
  • 更新版本：使用最新版本，通常有性能优化
  • 检查系统资源：确保系统有足够的可用内存

CPU 使用率过高怎么办？

+

降低 CPU 使用率的方法：

  • 更换协议：某些协议（如 VMess）CPU 占用较高，可尝试 VLESS 或 Trojan
  • 减少并发连接：在设置中降低最大并发连接数
  • 关闭不必要功能：如实时速度监控、详细日志等
  • 检查后台应用：确保没有其他应用占用过多 CPU

🔧 故障排除

应用无法启动怎么办？

+

解决启动问题的步骤：

  1. 检查系统兼容性：确认操作系统版本符合要求
  2. 以管理员权限运行：右键选择"以管理员身份运行"
  3. 检查防火墙：确保防火墙没有阻止应用
  4. 重新安装：卸载后重新下载安装最新版本
  5. 检查依赖：确保系统安装了必要的运行库
  6. 查看日志：检查应用日志文件获取错误信息

如何重置应用设置？

+

重置应用设置的方法：

方法一：应用内重置

  1. 打开 {{ .Site.Params.app.name }} 设置
  2. 找到"高级设置"或"系统设置"
  3. 点击"重置设置"或"恢复默认"

方法二：手动删除配置文件

  • Windows：%APPDATA%\HiddifyNext\
  • macOS：~/Library/Application Support/HiddifyNext/
  • Linux：~/.config/HiddifyNext/

注意：重置设置会删除所有配置，包括订阅信息，请提前备份重要数据。
